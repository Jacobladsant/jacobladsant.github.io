## 🇬🇧 ENGLISH VERSION
The Business Problem
Digital transformation is reshaping hiring strategies worldwide. But for professionals and companies deciding where to invest in training:
Which programming languages and databases will actually matter in the job market — today and in the near future?
What I Did
Analyzed three complementary data sources: Stack Overflow Developer Survey, real job postings data, and technology training portals. Combined qualitative survey data with quantitative job posting counts to identify skill gaps and emerging trends. Built visualizations in Python and a 3-tab Google Looker Studio dashboard covering current usage, future trends, and demographics.
Key Findings

🐍 Python leads future demand — while HTML/CSS/JavaScript dominates current usage, Python is the #1 language developers want to learn next, and commands a $114K average salary (second only to Swift)
🗄️ PostgreSQL is the database of the future — it dominates both current usage and future preference, ahead of MySQL and Microsoft SQL Server
☁️ AWS is the cloud platform to know — Amazon Web Services appears as the most demanded platform across job postings and developer surveys
📊 SQL appears in 310 job postings analyzed — confirming it remains a non-negotiable baseline skill for data roles
👥 The 25-34 age group represents the largest segment of tech professionals (41.1%), suggesting a highly competitive early-career market

Business Decision This Enables
An HR team or training department could use this analysis to prioritize Python, PostgreSQL, and AWS in their technical hiring criteria and upskilling programs — rather than investing equally across all technologies.

## 🇪🇸 VERSIÓN EN ESPAÑOL
El Problema de Negocio
La transformación digital está redefiniendo las estrategias de contratación a nivel global. Pero para los profesionales y empresas que deciden dónde invertir en capacitación:
¿Qué lenguajes de programación y bases de datos realmente importarán en el mercado laboral — hoy y en el futuro cercano?
Qué Hice
Analicé tres fuentes de datos complementarias: encuesta Stack Overflow Developer Survey, datos reales de ofertas de empleo, y portales de formación tecnológica. Combiné datos cualitativos de encuestas con conteos cuantitativos de vacantes para identificar brechas de habilidades y tendencias emergentes. Construí visualizaciones en Python y un dashboard de 3 pestañas en Google Looker Studio que cubre uso actual, tendencias futuras y demografía.
Hallazgos Clave

🐍 Python lidera la demanda futura — mientras HTML/CSS/JavaScript domina el uso actual, Python es el lenguaje #1 que los desarrolladores quieren aprender, con un salario promedio de $114K (segundo solo detrás de Swift)
🗄️ PostgreSQL es la base de datos del futuro — domina tanto el uso actual como la preferencia futura, por delante de MySQL y Microsoft SQL Server
☁️ AWS es la plataforma cloud a dominar — Amazon Web Services aparece como la plataforma más demandada en ofertas de empleo y encuestas de desarrolladores
📊 SQL aparece en 310 ofertas de empleo analizadas — confirmando que sigue siendo una habilidad base innegociable para roles de datos
👥 El grupo de 25-34 años representa el segmento más grande de profesionales tech (41.1%), lo que indica un mercado de entrada altamente competitivo

Decisión de Negocio que Habilita Este Análisis
Un equipo de RRHH o un departamento de capacitación podría usar este análisis para priorizar Python, PostgreSQL y AWS en sus criterios de contratación técnica y programas de upskilling — en lugar de invertir por igual en todas las tecnologías.
